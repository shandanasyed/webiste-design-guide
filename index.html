HTML Table Reference

HTML Elements

Tag Name Description
<table> Table Wraps around all HTML tables.
<thead> Table Head Defines the headings of table columns encapsulated in table rows.
<tbody> Table Body Contains all table data other than table heading and table footer content.
<tfoot> Table Footer Uses table rows to give footer content or to summarize content at the end of a table.
<th> Table Heading Adds titles to rows and columns of a table.
<tr> Table Row Adds rows to a table.
<td> Tabel Data Adds cell of data to a table (nested inside Table Row

HTML Attributes

Attribute Name Description
colspan Column Span Indicates how many column a cell spans in the table.
rowspan Row Span Inicates how many rows a cells spans in the table.
